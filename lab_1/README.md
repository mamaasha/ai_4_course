# Лабораторная лабора №1


**Тема:** Проведение исследований с алгоритмом KNN

## Данные

* **Классификация:** датасет с текстами спортивных новостей.
  Каждый текст относится к одному из 13 видов спорта (футбол, хоккей, теннис и др.).

* **Регрессия:** датасет о молочной продуктивности коров.
  Целевая переменная — *milk_yield_kg* (удой в кг).

## Этапы

1. **Выбор данных и метрик**
   Для классификации — `precision`, `recall`, `f1-score`.
   Для регрессии — `MAE`, `RMSE`, `R²`.

2. **Построение бейзлайна**
   Обучены базовые модели `KNeighborsClassifier` и `KNeighborsRegressor` из sklearn.
   Получены исходные метрики и проведён анализ ошибок.

3. **Улучшение бейзлайна**

   * Выполнен препроцессинг данных: очистка, нормализация, кодирование категориальных признаков.
   * Для классификации — балансировка классов (SMOTE) и подбор гиперпараметров (`k`, `weights`).
   * Для регрессии — удаление выбросов, масштабирование и подбор параметров.
   * Качество моделей существенно улучшилось.

4. **Имплементация алгоритма**
   Реализованы свои версии KNN для классификации и регрессии на NumPy.


## Итог

* Для классификации достигнута F1-score ≈ 0.79
* Для регрессии достигнуты метрики: **MAE ≈ 154**, **R2 ≈ 0.84**
