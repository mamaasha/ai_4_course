# Лабораторная лабора №2


**Тема:** Проведение исследований с логистической и линейной регрессией

---

### 2. Бейзлайн

* **Классификация:** датасет с текстами спортивных новостей.
  Цель — определить тему новости по тексту.

* **Регрессия:** датасет о молочной продуктивности коров.
  Цель — предсказать удой по характеристикам животного.


Использованы модели `LogisticRegression` и `LinearRegression` из sklearn. Это базовые версии для сравнения.

---

### 3. Улучшенный бейзлайн

Что было сделано:

* стандартизация признаков;
* добавление регуляризации (L1/L2);
* подбор гиперпараметров через `GridSearchCV`.
* предобработка данных

Результат:

* Для классификации F1 и Accuracy улучшились — модель стала точнее различать виды спорта.
* Для регрессии уменьшились ошибки MAE и RMSE — прогноз удоя стал надёжнее.
* Простые шаги по улучшению параметров заметно повысили качество.

---

### 4. Собственная реализация

* логистическая регрессия с градиентным спуском;
* линейная регрессия (аналитически и с градиентным спуском).
